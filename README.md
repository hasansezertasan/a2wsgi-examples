## How to run

- Requirements
  - Docker
  - Docker Compose
- Steps
  1. Clone the repository
  2. Go to the project directory
  3. Run `docker-compose up --build`
  4. Go to `http://localhost:8000/docs` to see the API documentation
  5. Enjoy!
